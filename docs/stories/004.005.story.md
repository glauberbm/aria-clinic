# User Story: CALE-005

**Epic:** EPIC-004 — Scheduler & Appointment Management
**Title:** WhatsApp Appointment Reminders & Notifications
**ID:** CALE-005
**Created:** 2026-05-15
**Status:** Draft
**Phase:** 2

---

## User Story Statement

**As a** clinic manager
**I want** to send WhatsApp reminders to patients before appointments
**So that** I can reduce no-shows and improve patient experience

---

## Acceptance Criteria

- [ ] Reminder settings page: Enable/disable, send time (24h before, 1h before, custom)
- [ ] Send button: Manually trigger reminders from appointment details
- [ ] Template editor: Customize reminder message with placeholders ({{PATIENT}}, {{DATE}}, {{TIME}}, {{DOCTOR}})
- [ ] Delivery status: Track if reminder sent (mock data shows "sent", "pending", "failed")
- [ ] Bulk send: Send reminders for all appointments on specific date
- [ ] No actual WhatsApp integration Phase 2 (mock send shows success)
- [ ] Delivery log: History of all reminders sent

---

## Technical Requirements

**Frontend:**
- Create `/app/scheduler/reminders/page.tsx` (settings)
- Create `/components/scheduler/ReminderSettings.tsx`
- Create `/components/scheduler/ReminderTemplate.tsx` (editor)
- Reminder options: Dropdown (24h before, 1h before, custom input)
- Message template: Text area with {{PLACEHOLDER}} examples
- Send simulation: Button click → 500ms simulated delivery → "Sent" status
- Mock data: 10 past reminders in log
- Bulk action: Select date range → send all reminders

**Styling:**
- Settings form: TailwindCSS
- Log table: Appointment date, patient, status, sent time
- Template editor: Text area with preview

**Testing:**
- Unit: Template placeholder replacement
- Integration: Reminder settings saved and applied
- Visual: Responsive on mobile/desktop
