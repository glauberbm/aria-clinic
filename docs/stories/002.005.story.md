# User Story: DASH-005

**Epic:** EPIC-002 — Dashboard
**Title:** Patient List — Display Recent Patients Table with Basic Info
**ID:** DASH-005
**Created:** 2026-05-15
**Status:** Draft
**Ready for:** @po validation

---

## Overview

Create a data table component displaying the 10 most recent patients with name, ID, phone, protocol, and last appointment date. Table includes sorting by columns and hover effects. Uses mock data, no external APIs (Phase 1).

---

## User Story Statement

**As a** clinic receptionist
**I want** to see a quick list of recent patients on the dashboard
**So that** I can quickly access patient information without navigating to a separate page

---

## Acceptance Criteria

- [ ] Patient table component created (`/components/dashboard/PatientTable.tsx`)
- [ ] Table displays: Patient Name, Patient ID, Phone, Protocol, Last Appointment
- [ ] Shows 10 most recent patients (mock data)
- [ ] Columns are sortable (click header to sort ascending/descending)
- [ ] Row hover effect: subtle background highlight
- [ ] Current sort indicator: up/down arrow on active column
- [ ] No pagination (displays fixed 10 rows for Phase 1)
- [ ] Responsive: Horizontal scroll on mobile (or stack layout), full grid on desktop
- [ ] Click row → navigate to patient details page (future story; stub link now)
- [ ] Accessible: Table with proper `<thead>`, `<tbody>`, aria-sort attributes on sortable headers
- [ ] No API calls (mock data only)

---

## Technical Requirements

**Frontend:**
- Create `/components/dashboard/PatientTable.tsx` (table component)
- Mock data structure:
  ```javascript
  const patientData = [
    {
      id: 'PT-001',
      name: 'Sarah Anderson',
      phone: '(555) 123-4567',
      protocol: 'Preventive Care',
      lastAppointment: '2026-05-14'
    },
    // ... 9 more patients
  ]
  ```
- Sorting: State hook `useState([sortField, sortOrder])` to track current sort
- Sort function: Array.sort() with comparator based on field type (string, date)
- Table markup: Semantic HTML `<table>`, `<thead>`, `<tbody>`
- Styling: TailwindCSS table classes + custom hover effects
- Row click: `useRouter()` to navigate to `/app/dashboard/patients/{id}` (stub route)

**Styling:**
- Table: w-full, border-collapse, background white with subtle border
- Header: bg-gray-100, font-bold, border-bottom
- Body rows: border-bottom (gray-200), hover:bg-gray-50
- Click cursor: Row pointer cursor on hover
- Responsive: On mobile (<768px), add horizontal scroll wrapper (`overflow-x-auto`)

**Sorting:**
- Click header → toggle sort ascending ↑ / descending ↓
- Default sort: By Last Appointment (most recent first)
- Sortable columns: Name, Protocol, Last Appointment (Phone/ID not sortable)
- Sort indicator: Arrow icon (↑ / ↓) in header cell

**Integration:**
- Add table to dashboard below charts
- Full width, positioned as bottom section

**Testing:**
- Unit test: Table renders with mock data
- Unit test: Sorting by different columns
- Integration test: Row click navigates to patient detail page (stub)
- Visual: Table layout on mobile (375px) with horizontal scroll, desktop full width
