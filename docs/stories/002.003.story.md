# User Story: DASH-003

**Epic:** EPIC-002 — Dashboard

**Status:** Draft
**Ready for:** @po validation

---

## Overview

Create a pie/donut chart component displaying the distribution of patients across different treatment protocols (e.g., Preventive Care, Acute Treatment, Chronic Management). Use mock data with color-coded segments and legend.

---

## User Story Statement

**As a** clinic analyst
**I want** to see a visual breakdown of patients by protocol type
**So that** I can understand patient distribution across different care categories

---

## Acceptance Criteria

- [ ] Protocol chart component created (`/components/dashboard/ProtocolChart.tsx`)
- [ ] Chart type: Donut (or pie) with color-coded segments
- [ ] Chart displays 4 protocol categories: Preventive Care, Acute Treatment, Chronic Management, Wellness
- [ ] Each segment shows: percentage and patient count on hover
- [ ] Legend positioned below or to the right of chart
- [ ] Total patient count displayed in donut center
- [ ] Mock data: no API calls
- [ ] Responsive: Chart scales to container on all screen sizes
- [ ] Colors: Distinct colors per protocol (accessible color palette)
- [ ] No loading states (static Phase 1)
- [ ] Accessible: Chart description in alt text, legend keyboard navigable

---

## Technical Requirements

**Frontend:**
- Create `/components/dashboard/ProtocolChart.tsx` (chart component)
- Chart library: **Recharts** (lightweight, React-friendly)
- Mock data structure:
  ```javascript
  const protocolData = [
    { name: 'Preventive Care', value: 156, color: '#3B82F6' },
    { name: 'Acute Treatment', value: 89, color: '#EF4444' },
    { name: 'Chronic Management', value: 67, color: '#10B981' },
    { name: 'Wellness', value: 30, color: '#F59E0B' }
  ]
  ```
- Chart: Responsive container with `<ResponsiveContainer>` from Recharts
- Tooltip: Show protocol name + count on hover
- Legend: Custom legend component below chart
- Center text: Display total (e.g., "342 Total Patients")

**Install Dependency:**
- Add `recharts` to `package.json` (if not already present)

**Styling:**
- Container: Fixed aspect ratio or min-height constraint (e.g., h-80)
- Legend: Display as grid or flex row below chart
- Colors: Accessible palette (avoid red/green color-blindness issues)
- Responsive: Works from 375px (mobile) to 1920px (desktop)

**Integration:**
- Add chart to dashboard main content area (below KPI cards or in side column)
- Grid layout: 1-column mobile, 2-column tablet/desktop (chart + financials)

**Testing:**
- Unit test: Chart renders with correct data
- Unit test: Hover tooltip displays correct value
- Integration test: Chart positioned correctly on dashboard
- Visual: Chart proportions on mobile/tablet/desktop
