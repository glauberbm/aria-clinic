## Overview

Implement patient dashboard providing comprehensive health overview including appointments, medical records, prescriptions, lab results, and health metrics in one personalized view.

---

## User Story Statement

**As a** patient
**I want** to see my complete health overview in one dashboard
**So that** I can easily manage my health and access important information

---

## Acceptance Criteria

- [ ] Patient dashboard at `/app/patient/dashboard`
- [ ] Upcoming appointments widget
- [ ] Recent medical records summary
- [ ] Active prescriptions display
- [ ] Lab results summary
- [ ] Health metrics (BMI, blood pressure history)
- [ ] Quick links to common actions (book appointment, view records)
- [ ] Health alerts/notifications
- [ ] Mobile-responsive design
- [ ] Customizable widgets

---

## Technical Requirements

**Frontend:**
- Create `/app/patient/dashboard` page
- Implement dashboard widgets (appointments, records, prescriptions)
- Health metrics visualization
- Responsive grid layout
- Real-time notifications

**Backend:**
- Endpoint: GET `/api/patient/dashboard-summary` (aggregate data)
- Endpoint: GET `/api/patient/health-metrics`
- Endpoint: GET `/api/patient/notifications`
- Aggregation logic for multiple data sources

**Database:**
- Query optimization for dashboard data aggregation
- Caching strategy for performance
