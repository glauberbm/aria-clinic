# User Story: DASH-004

**Epic:** EPIC-002 — Dashboard
**Title:** Financial Chart — Display Revenue Trend Over Time (Line Chart)
**ID:** DASH-004
**Created:** 2026-05-15
**Status:** Draft
**Ready for:** @po validation

---

## Overview

Create a line chart component showing clinic revenue trend over the past 12 months. Displays historical revenue data with grid lines, axis labels, and interactive tooltips. Uses mock data, no external APIs (Phase 1).

---

## User Story Statement

**As a** clinic financial manager
**I want** to see a line chart of monthly revenue trends
**So that** I can track financial performance and identify patterns

---

## Acceptance Criteria

- [ ] Financial chart component created (`/components/dashboard/FinancialChart.tsx`)
- [ ] Chart displays 12 months of mock revenue data
- [ ] Chart type: Line chart with area fill (gradient) under the line
- [ ] X-axis: Month labels (Jan, Feb, Mar, ..., Dec)
- [ ] Y-axis: Revenue in USD with currency formatting
- [ ] Tooltip: Shows month and revenue on hover
- [ ] Grid lines: Subtle background grid for readability
- [ ] Line styling: Bold line (2-3px) with smooth curve interpolation
- [ ] Responsive: Chart scales to container width on all screen sizes
- [ ] Legend: Shows "Monthly Revenue" label
- [ ] No loading states (static Phase 1)
- [ ] Accessible: Chart has descriptive title, axis labels, keyboard interaction support

---

## Technical Requirements

**Frontend:**
- Create `/components/dashboard/FinancialChart.tsx` (chart component)
- Chart library: **Recharts** (same as DASH-003)
- Mock data structure:
  ```javascript
  const revenueData = [
    { month: 'Jan', revenue: 18500 },
    { month: 'Feb', revenue: 19200 },
    { month: 'Mar', revenue: 21800 },
    { month: 'Apr', revenue: 20500 },
    { month: 'May', revenue: 22100 },
    { month: 'Jun', revenue: 24500 },
    { month: 'Jul', revenue: 23800 },
    { month: 'Aug', revenue: 25200 },
    { month: 'Sep', revenue: 26100 },
    { month: 'Oct', revenue: 24900 },
    { month: 'Nov', revenue: 27300 },
    { month: 'Dec', revenue: 28500 }
  ]
  ```
- Chart: `<LineChart>` with `<CartesianGrid>`, `<XAxis>`, `<YAxis>`, `<Tooltip>`, `<Area>`
- Tooltip: Custom formatter showing currency (USD) and month
- Area: Gradient fill under line (blue-ish color)
- Responsive container: Full width of parent
- Y-axis: Format numbers with $ prefix and K abbreviation (e.g., $20K)

**Styling:**
- Container: min-height of 400px, responsive width (100% of parent)
- Colors: Primary line color (blue-500 or brand color), area gradient (blue-100 to transparent)
- Line curve: Smooth interpolation (`type="monotone"`)
- Grid: Subtle gray (opacity 0.1)
- Responsive: Works from 375px (mobile) to 1920px (desktop)

**Integration:**
- Add chart to dashboard below KPI cards or in side column
- Grid layout: Pair with Protocol Chart (DASH-003) for 2-column layout on desktop

**Testing:**
- Unit test: Chart renders with correct data
- Unit test: Y-axis formatting (currency)
- Integration test: Chart positioned on dashboard
- Visual: Line curves correctly on mobile/tablet/desktop
