# User Story: DASH-001

**Epic:** EPIC-002 — Dashboard

**ID:** DASH-001
**Created:** 2026-05-15
**Status:** Draft
**Ready for:** @po validation

---

## Overview

Build the main dashboard shell with persistent header, responsive sidebar navigation, and main content area. Establish the layout foundation for all subsequent dashboard components.

---

## User Story Statement

**As a** clinic staff member
**I want** to see a professional dashboard layout with header and sidebar navigation
**So that** I can easily navigate between different dashboard sections and patient management features

---

## Acceptance Criteria

- [ ] Dashboard page created at `/app/dashboard` (Next.js App Router)
- [ ] Header component with clinic logo, user profile dropdown, and notifications icon
- [ ] Sidebar navigation with collapsible menu (mobile-responsive)
- [ ] Main content area with responsive grid layout (TailwindCSS)
- [ ] Mobile: Sidebar collapses to hamburger menu on screens < 768px
- [ ] Desktop: Sidebar always visible with fixed width (240px)
- [ ] Header sticky on scroll, sidebar scrollable independently
- [ ] Navigation links: Dashboard (home), Patients, Financials, Settings, Logout
- [ ] Active route highlighted in sidebar
- [ ] No external API calls (static layout only, Phase 1)

---

## Technical Requirements

**Frontend:**
- Create `/app/dashboard/layout.tsx` (Next.js layout component)
- Create `/components/dashboard/Header.tsx` (sticky header with profile menu)
- Create `/components/dashboard/Sidebar.tsx` (responsive navigation sidebar)
- Create `/app/dashboard/page.tsx` (main dashboard page placeholder)
- Use TailwindCSS for responsive grid (md: breakpoint for mobile → desktop transition)
- Use React state hook for sidebar collapse toggle (mobile)
- Navigation links via Next.js `<Link>` component
- Clinic logo: Placeholder from `/public/logo.svg` (or text-based)

**Styling:**
- Header: h-16 (64px) fixed at top
- Sidebar: w-64 fixed on desktop, collapsed on mobile
- Main content: ml-64 (desktop) / ml-0 (mobile), pt-16 (below header)
- Color scheme: From Tailwind defaults (gray/blue palette)
- Icons: Use `lucide-react` for hamburger menu, user profile, settings

**No APIs:**
- Static navigation structure
- No user data fetched
- No database calls
- Placeholder user name in profile dropdown

**Testing:**
- Unit tests: Sidebar collapse/expand on mobile
- Integration test: Navigation links render correctly
- Visual test: Layout structure on mobile (375px) and desktop (1920px)
