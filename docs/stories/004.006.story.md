# User Story: CALE-006

**Epic:** EPIC-004 — Scheduler & Appointment Management
**Title:** Waitlist Management — Manage Cancellations & No-shows
**ID:** CALE-006
**Created:** 2026-05-15
**Status:** Done
**Phase:** 2

---

## User Story Statement

**As a** clinic scheduler
**I want** to manage a waitlist of patients waiting for cancelled appointment slots
**So that** I can fill open slots and maximize clinic utilization

---

## Acceptance Criteria

- [x] Waitlist page at `/app/scheduler/waitlist` showing pending patients
- [x] When appointment cancelled: Option to add to waitlist automatically
- [x] Waitlist display: Patient name, requested date/time, time on list, priority
- [x] Manual offer: "Offer available slot" → sends WhatsApp offer to patient
- [x] Auto-fill option: Toggle to auto-assign next available slot
- [x] Patient response simulation: Mock accept/decline UI
- [x] Move to appointment: Confirmed acceptance creates appointment
- [x] No API calls (Phase 2)

---

## Technical Requirements

**Frontend:**
- Create `/app/scheduler/waitlist/page.tsx`
- Create `/components/scheduler/WaitlistForm.tsx` (add patient modal)
- Create `/components/scheduler/WaitlistOfferModal.tsx` (offer dialog)
- Waitlist state: Patient name, requested date/time, priority, status (pending, offered, accepted, declined)
- Offer flow: Select appointment slot → offer details modal → send (simulated)
- Response simulation: "Accept offer" button → auto-create appointment
- Sort: By time on list (FIFO) or priority (custom)

**Styling:**
- Waitlist table: Responsive on mobile/desktop
- Status badges: Different colors (pending, offered, accepted, declined)
- Modal for offers and acceptances

**Testing:**
- Unit: Waitlist order (FIFO or priority)
- Integration: Offer flow creates appointment
- Visual: Responsive table
