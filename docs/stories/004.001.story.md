# User Story: CALE-001

**Epic:** EPIC-004 — Scheduler & Appointment Management
**Title:** Calendar View — Display 30-Day Appointment Calendar
**ID:** CALE-001
**Created:** 2026-05-15
**Status:** Draft
**Phase:** 2 (Non-blocking for MVP)

---

## User Story Statement

**As a** clinic receptionist
**I want** to see a 30-day calendar view of all scheduled appointments
**So that** I can quickly find available time slots and manage the schedule

---

## Acceptance Criteria

- [ ] Calendar page created at `/app/scheduler` with Next.js App Router
- [ ] Displays current month + previous/next month navigation
- [ ] Each day cell shows appointment count or "free" indicator
- [ ] Dates clickable → shows appointments for that day (modal/sidebar)
- [ ] Timezone-aware (display user's local timezone, configurable)
- [ ] Highlight today's date with visual indicator
- [ ] Color-coded appointment types (consultation, followup, etc.)
- [ ] No API calls (Phase 2 integration with backend)
- [ ] Responsive: Works on mobile (375px) to desktop (1920px)
- [ ] Keyboard navigation: Arrow keys move between dates

---

## Technical Requirements

**Frontend:**
- Create `/app/scheduler/page.tsx` (main calendar page)
- Create `/components/scheduler/CalendarView.tsx` (calendar component)
- Library: `react-calendar` or custom grid layout with TailwindCSS
- Mock data: 15-20 sample appointments distributed across 30 days
- Navigation: Prev/Next month buttons
- Day click → show appointments in side panel or modal
- Timezone: Use browser's `Intl.DateTimeFormat` for local display

**Styling:**
- Grid layout: 7 columns (Sun-Sat) with day headers
- Color scheme: Green=available, Yellow=partial, Red=full
- Today highlight: Border or background color
- Responsive: Stack on mobile, full grid on desktop

**Testing:**
- Unit: Calendar renders correct month
- Unit: Day click shows appointments
- Integration: Month navigation works
- Visual: Responsive on 3 screen sizes
