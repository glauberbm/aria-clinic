**Created:** 2026-05-15
**Status:** Draft
**Phase:** 2

---

## User Story Statement

**As a** clinic scheduler
**I want** to assign appointments to doctors and see their availability
**So that** I can optimize schedule and prevent double-bookings

---

## Acceptance Criteria

- [ ] Doctor list page at `/app/scheduler/doctors` showing all doctors
- [ ] Each doctor card shows: name, specialty, availability status (available/busy)
- [ ] Doctor details modal: Schedule view (week/month), current appointments
- [ ] Assignment: Drag-drop appointment to doctor (or select in form)
- [ ] Availability: Manual set working hours (mock data, no persistence)
- [ ] Double-booking prevention: System prevents appointment overlap
- [ ] No API calls (Phase 2)
- [ ] Mock data: 5 doctors with working hours

---
